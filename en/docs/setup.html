3. Setup

3.1. Setting up FF-SDK-Unity

3.1.1. Import unitypackage

  1. From the menu in the Unity Editor, select Assets / Import Package / Custom Package... and import .unitypackage.
  2. If the installation dialog for the TextMesh Pro asset appears, install it.

The types of unitypackages and the description

unitypackage Description
FF-SDK-Unity_Core FF-SDK-Unity core features
It is required to import for SDK operation.
FF-SDK-Unity_Windows Features for FeelFuse with USB Wired Connection
Please also complete the setup described in
3.5. Setting up FeelFuse with USB Wired Connection (Optional)
FF-SDK-Unity_MetaXR Import it when using the connection feature with Meta Quest
Please also complete the setup described in
3.3. Setting up Meta XR SDK (Optional)
FF-SDK-Unity_TofAR Import it when using the features that supporting development and debug connected with ToF AR
Please also complete the setup described in
3.4. Setting up ToF AR SDK (Optional)
FF-SDK-Unity_Samples Sample scenes

3.1.2 Import OscCore unitypackage

  1. Download OscCore_vX.X.X.unitypackage from https://github.com/stella3d/OscCore/releases, and import it.

3.2. Upgrade FF-SDK-Unity

  1. Delete Commissure/HapticsSystem folder in the Project view of the Unity Editor.
  2. From the menu in the Unity Editor, select Assets / Import Package / Custom Package... and import the .unitypackage file.

3.3. Setting up Meta XR SDK (Optional)

By using the Meta XR SDK, you can develop an application that enables haptic experiences with head-mounted displays such as Meta Quest.

If you use the Meta XR SDK, please follow the Meta XR SDK setup steps.
The use of the Unity OpenXR plugin is recommended for XR plugins.

https://developers.meta.com/horizon/documentation/unity/unity-project-setup

3.4. Setting up ToF AR SDK (Optional)

By using ToF AR SDK and supported smartphones, it allows to to streamline development and debugging without wearing a head-mounted display.

  1. Download ToF AR SDK from below.
    https://tof-ar.com/
  2. Import TofAR_Base_vx.x.x_Android.unitypackage and TofAR_Base_vx.x.x_iOS.unitypackage.

No changes to modify Unity Project settings since Deployment to a smartphone is not required.

3.5. Setting up FeelFuse with USB Wired Connection (Optional)

To run FeerFuse with USB Wired Connection, install assets below:

  • AudioStream (The asset requires a fee.)
    https://assetstore.unity.com/packages/tools/audio/audiostream-pcm-audio-for-unity-65411

  • FMOD for Unity
    https://assetstore.unity.com/packages/tools/audio/fmod-for-unity-2-02-161631
